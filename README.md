# TypeScript Examples

## Set up

```
yarn install
```

## Run

```bash
# Run index.js with
yarn start
# Start server with
yarn start:server
```

## What is this?

Project highlighting different aspects of TypeScript

Including:
- Basic project structure
    - [tsconfig](https://www.typescriptlang.org/tsconfig)
- Basic Types
- Function Types
- Interfaces
- Literal Types
- Union Types
- Using a library
